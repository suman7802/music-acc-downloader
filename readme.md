Download YouTube's music/playlist on AAC format

create .env file
Get YouTube_API_KEY from https://console.cloud.google.com/

- using Node
  npm run start

- Using Docker Container
  docker-compose up --build

- Now you are done
  server http://localhost:8000
  client http://localhost:5173

- Navigate to
  http://localhost:5173
